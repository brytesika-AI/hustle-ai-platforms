# Hustle Workforce

Hustle Workforce is the flagship product in the Hustle AI suite. It is positioned as an executive-grade workforce intelligence platform for businesses that need sharper visibility into labor performance, productivity, scheduling pressure, and operating efficiency.

## Product Role

This product sets the standard for the overall portfolio. It should lead the narrative in repository materials, product demos, and early build sequencing.

## Product Scope

- Workforce planning visibility
- Productivity and performance intelligence
- Labor utilization insight
- Management reporting surfaces
- Operational intervention support

## Product Structure

This folder is intended to remain self-contained while reusing shared modules where they improve speed and consistency.

Suggested internal areas:
- `app/` for product implementation
- `docs/` for product-specific documentation
- `config/` for runtime and deployment configuration
