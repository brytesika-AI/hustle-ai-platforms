# Product Checklist

- Cloudflare-first web application architecture is in place
- Next.js frontend scaffold is implemented
- Python backend API is implemented
- Executive, finance, operations, sales, strategy, growth, knowledge brain, and about sections are implemented
- Cashflow, inventory, WhatsApp CRM, decision copilot, and marketing campaign agents are implemented
- CSV and text upload support is implemented through the backend analyze route
- Synthetic demo data is available when no uploads are provided
- Business health engine is implemented
- Knowledge ingest, batch ingest, query, lint, explore, and brief generation flows are implemented
- Persistent department-agent architecture is represented through product state and artifact paths
- Decision workflows are positioned as multi-step and continuation-ready
- Knowledge-brain artifacts are expected to persist as briefs, wiki pages, and summaries
- Async-ready management digests and review cycles are part of the specification
- Qwen 2.5 Instruct is the default for executive summaries, multi-step reasoning, and decision copilots
- Mistral Small Instruct is the default for routine summaries, parsing, and issue extraction
- bge-m3 is the default for SME Knowledge Brain retrieval and semantic search
- Backend Docker packaging is included
- Cloudflare deployment guidance is included
