# Hustle Workforce Deployment

Deploy the frontend to Cloudflare Pages, run the FastAPI backend as a containerized service, and publish the backend through Cloudflare Tunnel or an equivalent Cloudflare-protected origin. Set `NEXT_PUBLIC_API_BASE_URL`, `CLOUDFLARE_API_TOKEN`, `CLOUDFLARE_ACCOUNT_ID`, and `CLOUDFLARE_ZONE_ID` in the deployment environment.
