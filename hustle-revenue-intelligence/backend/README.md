# Hustle Revenue Intelligence Backend

Python backend for pricing analysis, revenue driver modeling, churn-impact estimation, upsell and cross-sell signal detection, and executive revenue brief generation.

Run locally with:

```bash
pip install -r backend/requirements.txt
uvicorn backend.app.main:app --reload --host 127.0.0.1 --port 8000
```
