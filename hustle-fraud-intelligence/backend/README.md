# Hustle Fraud Intelligence Backend

Python backend for anomaly detection, fraud-risk signal review, supplier and expense red-flag analysis, refund-abuse monitoring, governance brief generation, and Cloudflare-only model configuration.

Run locally with:

```bash
pip install -r backend/requirements.txt
uvicorn backend.app.main:app --reload --host 127.0.0.1 --port 8000
```
