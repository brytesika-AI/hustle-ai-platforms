# Hustle Fraud Intelligence

Hustle Fraud Intelligence is a fraud detection product built for anomaly monitoring, suspicious activity analysis, case prioritization, and business trust protection.

## Product Scope

- Fraud pattern detection
- Transaction anomaly monitoring
- Investigation prioritization
- Protective reporting and alerts

## Product Structure

This product remains self-contained and may reuse shared modules for interface patterns, utilities, model routing, styles, and deployment conventions.
