# Hustle Inventory Intelligence Deployment

Deploy the frontend through Cloudflare Pages, run the backend in Docker, and attach the backend to a Cloudflare Tunnel route. Configure the deployment with `NEXT_PUBLIC_API_BASE_URL`, `CLOUDFLARE_API_TOKEN`, `CLOUDFLARE_ACCOUNT_ID`, and `CLOUDFLARE_ZONE_ID`.
