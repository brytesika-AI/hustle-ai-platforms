# Hustle CX Intelligence Backend

Python backend for transcript analytics, sentiment and severity analysis, churn risk scoring, root-cause clustering, product-signal detection, and executive CX briefing generation.

Run locally with:

```bash
pip install -r backend/requirements.txt
uvicorn backend.app.main:app --reload --host 127.0.0.1 --port 8000
```
