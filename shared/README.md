# Shared Modules

The `shared/` layer contains reusable building blocks that support consistency, speed, and engineering discipline across the Hustle AI portfolio.

It is intended for cross-product assets that are genuinely reusable across multiple products.

Included shared domains:
- `components/` for reusable UI patterns
- `utils/` for common helpers and business-safe utility logic
- `models/` for provider-agnostic model routing foundations
- `styles/` for shared tokens and presentation standards
- `cloudflare/` for deployment templates and release conventions
